## 🚀 Getting Started

### Prerequisites

- [.NET 8 SDK](https://dotnet.microsoft.com/download)

### Clone & Run

```bash
git clone https://github.com/amritsyangtan-sudo/Screen-Break.git
cd Screen-Break
dotnet run
```

### Build

```bash
dotnet build
```

### Publish (Windows self-contained)

```bash
dotnet publish -c Release -r win-x64 --self-contained
```
